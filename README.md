# 🍋 Little Lemon  

**Little Lemon** is a modern and responsive web application developed as part of the **Meta Front-End Developer Capstone Project**. It represents a family-owned Mediterranean restaurant, combining traditional recipes with a contemporary digital experience.  

## ✨ Key Features  

- **Responsive & Accessible UI** – Optimized for all screen sizes and user-friendly interactions.  
- **Online Table Reservations** – Enables customers to book tables conveniently.  
- **Dynamic Menu Display** – Showcases the restaurant’s offerings with an engaging interface.  
- **Optimized Performance** – Built with Vite for fast development and efficient bundling.  

## 🚀 Getting Started  

### Prerequisites  

Ensure you have the following installed:  
- **Node.js** (Latest LTS version recommended)  
- **npm** or **yarn** for package management  

### Installation  

1. Clone the repository:  

   ```bash
   git clone https://github.com/gihanchamila/Little-Lemon.git
   cd Little-Lemon
   ```

2. Install dependencies:  

   ```bash
   npm install
   ```

3. Start the development server:  

   ```bash
   npm run dev
   ```

   The application will be available at **`http://localhost:3000`**.  

## 🛠️ Technologies Used  

- **React.js** – Component-based UI development  
- **Vite** – Fast build tool for modern web projects  
- **CSS Modules** – Scoped and maintainable styling  
- **React Router** – Navigation management  

## 📂 Project Structure  

- `src/` – Main application source code  
  - `api/` – apis for app
  - `assets/` – Static resources such as images and icons  
  - `components/` –  Reusable UI components
